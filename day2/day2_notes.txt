# Node.js Backend Fundamentals Notes

## 1. Why Node.js?

Node.js allows JavaScript to run outside the browser.

Before Node.js:

```text
JavaScript → Browser Only
```

After Node.js:

```text
JavaScript
    |
    +-- Browser
    |
    +-- Server
```

This allows us to build backend applications using JavaScript.

---

# 2. What is HTTP?

HTTP stands for:

```text
HyperText Transfer Protocol
```

It is the communication protocol used between:

```text
Client (Browser)
        ↕
Server
```

Example:

```text
Browser Request:
GET /about

Server Response:
About Page
```

---

# 3. What is a Server?

Definition:

A server is a program that listens for requests and sends responses.

Mental Model:

```text
Restaurant

Customer → Order Food
Restaurant → Process Order
Restaurant → Send Food
```

Backend Equivalent:

```text
Client → Request
Server → Process Request
Server → Response
```

---

# 4. HTTP Module

Definition:

The built-in Node.js module used to create HTTP servers and handle HTTP requests/responses.

Import:

```js
const http = require("http");
```

Mental Model:

```text
fs module   → Talk to files

path module → Talk to paths

http module → Talk to internet
```

# 5. Creating a Server

Syntax:

```js
const http = require("http");

const server = http.createServer((req, res) => {

});
```

Definition:

```js
http.createServer(callback)
```

creates a server and registers a callback that runs whenever a request arrives.

Mental Model:

```text
Server Created
      |
      V
Waiting...
      |
      V
Request Arrives
      |
      V
Callback Executes
```

# 11. Why Browser Keeps Loading

Code:

```js
const server = http.createServer((req, res) => {
    console.log("Request Received");
});
```

Problem:

```text
Request received
Response never sent
```

Result:

```text
Browser keeps loading forever
```

Fix:

```js
res.end("Done");
```

---

# 12. What is Routing?

Definition:

Routing is deciding what logic should run based on the URL.

Example Routes:

```text
/
/about
/contact
```

Mental Model:

```text
Hotel

Route = Room Number
```

---

# 13. Basic Routing

Code:

```js
const server = http.createServer((req, res) => {

    if(req.url === "/"){
        res.end("Home Page");
    }

    else if(req.url === "/about"){
        res.end("About Page");
    }

    else{
        res.end("404 Page Not Found");
    }

});
```

Mental Model:

```text
Request Arrives
       |
       V
req.url
       |
       V
Choose Logic
       |
       V
Send Response
```

# 15. Listener

Definition:

A callback function waiting for an event.

Syntax:

```js
emitter.on("login", () => {
    console.log("User Logged In");
});
```

Meaning:

```text
If login event happens,
run this callback.
```

# 17. Complete EventEmitter Example

```js
const EventEmitter = require("events");

const emitter = new EventEmitter();

emitter.on("login", () => {
    console.log("Welcome User");
});

emitter.emit("login");
```

Output:

```text
Welcome User
```

# 19. EventEmitter + HTTP Server

Listener:

```js
emitter.on("requestReceived", (req) => {
    console.log(`Route Visited: ${req.url}`);
});
```

Server:

```js
const server = http.createServer((req, res) => {

    emitter.emit("requestReceived", req);

    res.end("Done");

});
```

Flow:

```text
Request Arrives
       |
       V
emit("requestReceived")
       |
       V
Listener Executes
       |
       V
Response Sent
```

---

# 20. Request Counter

Code:

```js
let requestCount = 0;

emitter.on("requestReceived", () => {
    requestCount++;

    console.log(`Total Requests: ${requestCount}`);
});
```

Why Doesn't It Reset?

Because:

```js
let requestCount = 0;
```

runs only once when:

```bash
node server.js
```

starts.

Mental Model:

```text
Server Process
      |
      +-- requestCount
      +-- EventEmitter
      +-- HTTP Server
      |
      V
Requests Keep Arriving
```
