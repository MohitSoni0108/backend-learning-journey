Manipulate HTML

Update the UI

It could not:

Read files
Create servers
Access OS information
Work with databases

But developers had to learn another language for backend
So the question became:

"Can JavaScript run outside the browser?

The Solution: Node.js ->Instead of running inside a browser: runs directly on your computer.

Why Is Node.js So Important For Backend?

Backend needs to:

Receive requests
Talk to databases
Read/write files
Authenticate users
Send emails
Handle payments

Node gives JavaScript access to all these capabilities


Node.js is a JavaScript runtime environment built on Google's V8 engine that allows JavaScript to run outside the browser and interact with the operating system, files, networks, and databases.
